# Student Performance Prediction System

# Project Overview

This project is a Machine Learning based Student Performance Prediction System developed using **Python**, **Scikit-learn**, and **Streamlit**.

The application predicts the student's final performance based on different input features such as study time, attendance, sleep hours, parental education, internet access, extracurricular activities, part-time job, previous grade, and final exam score.

---

# Technologies Used

- Python
- Pandas
- Scikit-learn
- Streamlit
- Joblib
